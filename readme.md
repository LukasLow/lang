# Wetter App in allen sprachen


----
### Python
```bash
cd python && py wetter.py && cd ..
```
```bash
cd python
py wetter.py
```

### Go
```bash
cd Go && go run main.go && cd ..
```
```bash
cd Go
go run main.go
```

### C
```bash
cd C && gcc -o wetter wetter.c -lcurl && ./wetter && cd ..
```
```bash
cd C
gcc -o wetter wetter.c -lcurl 
./wetter
cd ..
```

### Rust
```bash
cd rust && cargo run && cd ..
```
```bash
cd rust
cargo run
cd ..
```
